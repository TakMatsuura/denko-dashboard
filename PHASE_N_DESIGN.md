# 📊 Phase N: 多年度分析ダッシュボード 設計書

ver0.1 / 2026-05-03 / 作成: 松浦 + Claude / 対象: DENKO Dashboard 拡張

---

## 1. 背景・目的

Phase L (Phase M) で過去FY (FY2020〜FY2025) のデータが揃った。
これを活用して **長期トレンド分析・年次比較・顧客ライフサイクル** を見える化する。

---

## 2. 機能一覧

### N-1. 📊 年度比較サマリータブ
- 全FY (FY20〜FY26) の年間サマリーを **1画面** で並列表示
- カード形式で各年度の: 売上 / 粗利 / 利益率 / 仕入 / 件数
- 年度クリックで詳細展開 (drill-down)

### N-2. 👥 顧客年次推移
- 主要顧客 (Top 20) の年ごとの売上推移
- 折れ線グラフ (横軸: FY / 縦軸: 売上) - 顧客ごと
- フィルター: 顧客選択 / 部門選択

### N-3. 📈 YoY 成長率分析
- 前年比成長率を自動計算 (色分け表示)
- 全体・部門別・顧客別の3階層
- ヒートマップ表示

### N-4. ⏱️ 長期トレンド
- FY20/05〜FY26/04 の **72ヶ月連結** 月次時系列
- 売上・粗利・利益率の3軸折れ線
- 期末/期初マーカー (FY境界の縦線)

### N-5. 🔄 顧客ライフサイクル
- 取引開始FYからの累計売上推移
- 離脱顧客の検出 (直近2年取引なし)
- 新規獲得顧客の検出

---

## 3. 実装方針

### 3-1. データソース
既存の `public/data/fy/{year}/*.csv` を fetch で全FY ロード。

### 3-2. アーキテクチャ
- **Frontend のみで完結** (JavaScript で全FY データを集計)
- Cloudflare Pages Functions は不要 (静的ファイル配信のみ)
- 初回ロード時に全FY のCSV (約 8MB / 6FY × 1.3MB) を fetch → ブラウザにキャッシュ

### 3-3. 新タブ追加
既存タブ: Overview / Budget / Customers / Orders / Purchasing / Master
↓
**新規追加**: Multi-Year (or 多年度分析) タブ

### 3-4. 既存機能との互換性
- 年度フィルター (?fy=YYYY) はそのまま動作
- 多年度分析タブのみ全FY 横断
- 他のタブは現状の単一FY 表示

## 4. ファイル変更箇所

| ファイル | 変更内容 |
|---|---|
